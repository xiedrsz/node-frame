## 目录结构

* bin --------------------- 程序执行文件目录
* node_modules ---- 本工程所依赖的模块
* routes ---------------- 路由配置
* server ---------------- 后台服务器目录
* www ------------------ 前端代码
* tools ------------------ 工具包，用户自动化构建工程
* app.js ---------------- Express工程主文件
* package.json ------ nodejs工程配置
* readme.md --------- 说明文档
